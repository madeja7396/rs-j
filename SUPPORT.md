# Support

## Where to Ask Questions

- Usage questions and troubleshooting:
  https://github.com/madeja7396/rs-j/discussions
- Bug reports:
  https://github.com/madeja7396/rs-j/issues/new/choose

## Before Opening an Issue

- Check `README.md` and `docs/` first.
- Search existing issues/discussions.
- Include OS, terminal, app version (`btm -V`), and reproduction steps.

## Security Issues

For security-related reports, follow `SECURITY.md` and do not use public issues.
